Pythonゲーム紹介：「あっち向いてホイ」

🎮ゲームの概要

このゲームは、Pythonを使って作成した「あっち向いてホイ」です。
変数・条件分岐（if文）・キーボード入力（input）・乱数（random）などの基本機能を組み合わせて構築しました。
プレイヤーは「上・下・左・右」のいずれかを入力し、コンピュータの出した方向と一致したら負けとなります。

🧠使ったスキル

  • 変数の定義と使用
  • 条件分岐による判定処理
  • キーボード入力によるユーザー操作
  • 乱数によるコンピュータの動作決定

💻ゲームのソースコード

import random

print("あっち向いてホイ ゲーム！")
print("上(u)、下(d)、左(l)、右(r)のどれかを入力してね！")

directions = ["u", "d", "l", "r"]

while True:
    user = input("あなたの向き（u/d/l/r）→ ")
    if user not in directions:
        print("正しく入力してください（u, d, l, r）")
        continue

    comp = random.choice(directions)
    print("コンピュータの向き：", comp)

    if user == comp:
        print("あっち向いてホイ！負け！")
        break
    else:
        print("セーフ！もう一回！")

✨まとめ

Pythonの基本構文を使って、簡単ながらも対戦感のあるゲームを作ることができました。
コンピュータの乱数処理を利用することで、毎回違う結果になる面白さがあります。
