Flaskについて

Flaskとは？

FlaskはPythonでWebアプリケーションを作ることができる軽量なフレームワークです。 シンプルな構造で初心者でも扱いやすく、自分で自由に機能を追加できるのが特徴です。

今回作ったプログラム

CSVファイルを読み込み、そのデータをWebページに表示するプログラムを作成しました。 特に、場所のデータをGoogleマップのリンクに変換する工夫をしました。

工夫した点

  • CSVのデータをリストとして取得
  • 位置情報をGoogleマップURLに変換
  • 条件分岐で表示方法を変更

データ表示

{% for data in mmlist %} {% if data[1] == "場所" %} {{ data[1] }}
{% elif data|length > 3 and data[3] %} {{ data[1] }}
{% else %} {{ data[1] }}
{% endif %} {% endfor %}

まとめ

Flaskを使うことで、Pythonのデータ処理とWebページを簡単に連携させることができました。 今後はデザインの向上や、データベースとの連携などにも挑戦していきたいです。
