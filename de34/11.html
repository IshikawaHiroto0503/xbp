たたき起こし

朝、明るくなると自動で起こしてくれる装置

作品の概要

この作品は、朝になり明るくなったタイミングでモーターが動き、 お母さんがたたき起こしてくれるような動作を再現した装置です。 光センサーを利用し、一定以上の明るさになるとモーターが作動する仕組みになっています。

工夫した点

  • モーターの位置と子供の位置を調整し、しっかり起こせるようにした
  • 明るさのしきい値を設定し、朝だけ動くようにした
  • 暗いときにはモーターが完全に停止するようにプログラムを改良した

動作動画

Arduinoコード

const int sensorPin = A0;
int threshold = 500;

const int DIR  = 8;
const int STEP = 9;

void setup() {
  pinMode(DIR, OUTPUT);
  pinMode(STEP, OUTPUT);

  digitalWrite(DIR, LOW);
  digitalWrite(STEP, LOW);

  Serial.begin(9600);
}

void loop() {

  int val = analogRead(sensorPin);
  Serial.println(val);

  // ⭐ 明るいときだけ1ステップずつ動かす
  if (val >= 750) {

    // 時計回り
    digitalWrite(DIR, HIGH);
    stepMotor(5000);

  } else {
    // ⭐ 完全停止
    stopMotor();
  }
}

// 1ステップだけ動かす
void stepMotor(int delaytime) {
  digitalWrite(STEP, HIGH);
  delayMicroseconds(delaytime);
  digitalWrite(STEP, LOW);
  delayMicroseconds(delaytime);
}

// 完全停止
void stopMotor() {
  digitalWrite(STEP, LOW);
}

まとめ

光センサーとモーターを組み合わせることで、 明るさに応じて動作する装置を作ることができました。 実際の生活にも応用できる仕組みであり、 センサー制御の仕組みについて理解を深めることができました。
